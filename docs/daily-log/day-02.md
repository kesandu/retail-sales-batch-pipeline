## 4. Recall questions

**What is Azure?**  
Azure is Microsoft’s cloud platform. It gives me services over the internet for things like storage, computing, networking, and data work without needing to run everything on my own machine.

**What is a subscription?**  
A subscription is the top-level billing and management boundary in Azure. It controls ownership, access, and charging for the resources inside it.

**What is a resource group?**  
A resource group is a logical container that keeps related Azure resources together so they can be managed as one unit.

**What is a storage account?**  
A storage account is the Azure resource that provides access to storage services such as blobs, files, queues, and tables.

**Why does a storage account matter in this project?**  
A storage account matters because the project will need a cloud place to hold data files. In a data project, it can be the home for raw inputs, processed files, and other stored project data.
