# Retail Sales Batch Pipeline

A credible, production-minded batch data engineering project built to simulate a client-facing retail sales reporting workflow.

## Project goal

This project will ingest raw retail sales files, validate and transform the data, and produce business-friendly outputs with clear operational logic and failure handling.

## Planned scope

- ingest raw sales files
- clean and validate data
- generate processed outputs
- document business rules and pipeline flow
- include basic failure handling and checks

## Repository structure

- `data/raw/` - source input files
- `data/processed/` - cleaned outputs
- `docs/` - project documentation
- `notebooks/` - optional exploration
- `src/` - pipeline code
- `tests/` - validation and test logic
